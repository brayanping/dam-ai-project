# Instrucciones personalizadas para Copilot

## Título de conversación
Cuando el usuario haga su primera pregunta en una conversación, extrae automáticamente el tema principal y usa una versión corta y descriptiva como título sugerido para el historial de conversaciones.

Por ejemplo:
- Si pregunta "¿cómo arreglo el bat?", sugiere: "Arreglar archivo bat"
- Si pregunta "cambiar el color de la interfaz", sugiere: "Cambiar colores UI"
- Si pregunta "error al conectar con la base de datos", sugiere: "Error conexión BD"

Mantén los títulos concisos (3-8 palabras), en español, y descriptivos para facilitar la búsqueda en el historial.
